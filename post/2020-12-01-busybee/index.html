busybee

Data-Driven Productivity Tool

 Posted on December 1, 2020

Full-stack, interactive web application to track your goals, subtasks, and completion rates in real time. Check it out on Google Cloud Platform:

Description

busybee allows users to organize their lives into categories: school, work, hobbies, etc. They can then create goals within those categories and subtasks in checklist format. As they create goals and complete tasks, their usage data is stored and reflected in a basic dashboard on the homepage, which shows:

  1. Tasks completed vs. incomplete
  2. Breakdown of categories by number of goals in each, with the option to drill down into a given category
  3. Tasks completed over time, offering a basic measure of user consistency
